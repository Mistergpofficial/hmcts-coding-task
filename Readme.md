# HMCTS Task Management System

This project is a simple task management system built for the HMCTS Developer Coding Challenge.  
It allows caseworkers to create, view, update, and delete tasks through a clean API and a lightweight frontend interface.

The system is built using Laravel for the backend and Blade with vanilla JavaScript for the frontend.

---

## Features

### Backend API
- Create a task  
- Retrieve all tasks  
- Retrieve a task by ID  
- Update a task  
- Update task status  
- Delete a task  
- Validation and error handling  
- Database storage  
- Feature tests included  

### Frontend
- Create tasks  
- View tasks in a table  
- Edit tasks  
- Delete tasks  
- Simple and user friendly interface  
- No build tools required  

---

## Tech Stack

**Backend:** Laravel  
**Frontend:** Blade + Vanilla JavaScript  
**Database:** MySQL (or any Laravel supported database)  
**Testing:** Laravel Feature Tests
